# Client Handoff

## Handoff Package Contents

1. **Live URLs** — website, dashboard, API health endpoint
2. **Login credentials** — dashboard access (delivered via magic link)
3. **Client documentation** (generated as `output/client-docs.md`)
4. **Training session** (30-minute video call)
5. **Support SLA** — response times, included revisions, pricing for additions

## Training Session Agenda (30 min)

| Time | Topic |
|------|-------|
| 0–5 min | Website walkthrough: all pages, confirm everything looks correct |
| 5–15 min | Dashboard walkthrough: KPI cards, charts, funnel visualization |
| 15–20 min | Metrics glossary: what each number means for their business |
| 20–25 min | Reports overview: when reports arrive, how to read them |
| 25–30 min | Q&A + support process: how to reach PixelPro |

## Support SLA

| Tier | Included | Response Time |
|------|----------|---------------|
| Standard (included) | Bug fixes, analytics issues, report errors | 24h business days |
| Content revisions | Text updates, image swaps, minor copy changes | 48h; 2 included/month (first 3 months) |
| Feature additions | New pages, new KPIs, new integrations | Quoted separately |
| Emergency (site down) | Full site inaccessible | 4h; text/call directly |
