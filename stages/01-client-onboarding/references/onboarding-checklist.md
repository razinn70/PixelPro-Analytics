# Client Onboarding Checklist

## 1. Business Information
- [ ] Business name (legal + trading name)
- [ ] Business vertical: restaurant / ecommerce / service
- [ ] Owner name and contact email
- [ ] Primary phone number
- [ ] Business address (for local SEO)
- [ ] Business hours (restaurant/service)
- [ ] Existing website URL (if any)
- [ ] Existing social media accounts

## 2. Technical Inventory
- [ ] Current hosting provider
- [ ] Domain registrar (e.g., GoDaddy, Namecheap, Google Domains)
- [ ] Does client own their domain? If not, purchase needed.
- [ ] Existing Google account for GA4 setup
- [ ] Existing analytics tool (if any — note for migration)
- [ ] E-commerce platform (Shopify, WooCommerce, custom)
- [ ] Payment processor (Stripe, Square, PayPal)
- [ ] Email marketing tool (Mailchimp, Klaviyo, etc.)

## 3. Goals and KPIs
- [ ] Primary business goal (more orders / more leads / more bookings)
- [ ] Current monthly revenue (optional, for baseline)
- [ ] Current monthly website visitors (if known)
- [ ] Biggest pain point (cart abandonment / low traffic / no visibility)
- [ ] Target KPIs: choose minimum 5 from metric-definitions.md

## 4. Content and Design
- [ ] Brand logo (SVG or high-res PNG)
- [ ] Brand color hex codes (primary, secondary, dark)
- [ ] Font preferences (or use PixelPro defaults)
- [ ] Existing photography / product images
- [ ] Copy: tagline, about, service descriptions, menu items
- [ ] Competitor sites for design reference

## 5. Scope Definition
- [ ] Page list confirmed (from vertical-templates.md defaults + client additions)
- [ ] Feature list confirmed (ordering, contact form, booking, product catalog, etc.)
- [ ] Analytics features confirmed (funnel steps, custom events)
- [ ] Dashboard KPIs confirmed (minimum 5)
- [ ] Out-of-scope items documented

## 6. Agreements
- [ ] Project scope signed or approved in writing
- [ ] Timeline communicated
- [ ] Payment terms agreed
- [ ] Support SLA communicated
- [ ] Data privacy notice delivered (PIPEDA compliance)
