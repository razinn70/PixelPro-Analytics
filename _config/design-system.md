# PixelPro Design System

## Color Tokens

| Token | Hex | Usage |
|-------|-----|-------|
| `--color-primary` | #4A90D9 | CTAs, links, active states |
| `--color-accent` | #FF6B35 | Highlights, badges, alerts |
| `--color-dark` | #0F172A | Dark backgrounds, nav |
| `--color-surface` | #1E293B | Cards, panels |
| `--color-muted` | #64748B | Secondary text |
| `--color-text` | #F8FAFC | Primary text on dark |
| `--color-border` | #334155 | Dividers, outlines |
| `--color-success` | #22C55E | Positive trends, pass states |
| `--color-danger` | #EF4444 | Errors, negative trends |

## Typography

| Scale | Size | Weight | Usage |
|-------|------|--------|-------|
| `text-display` | 48px | 700 | Hero headings |
| `text-h1` | 36px | 700 | Page titles |
| `text-h2` | 28px | 600 | Section headings |
| `text-h3` | 22px | 600 | Card headings |
| `text-body` | 16px | 400 | Body copy |
| `text-small` | 14px | 400 | Labels, captions |
| `text-mono` | 14px | 400 | Code, metrics |

Font family: `Inter` (UI), `JetBrains Mono` (code/data)

## Spacing Scale

Base unit: 4px. Use multiples: 4, 8, 12, 16, 24, 32, 48, 64, 96.

## Component Tokens

- Border radius: `rounded-lg` (8px) for cards, `rounded-full` for badges
- Shadow: `shadow-lg` for elevated cards, `shadow-none` for flat panels
- Transition: `duration-200 ease-out` for all interactive states

## Breakpoints

| Name | Width | Usage |
|------|-------|-------|
| `sm` | 320px | Minimum supported |
| `md` | 768px | Tablet / dashboard minimum |
| `lg` | 1024px | Desktop default |
| `xl` | 1440px | Wide desktop |
